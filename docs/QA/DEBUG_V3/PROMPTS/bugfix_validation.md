# Bugfix Validation Prompt (T-0101)

Validate the proposed fix using the provided handoff zip/RAW links and respond in Markdown:
- Summary of Fix Under Test
- Validation Steps (commands / pages checked)
- Results: Pass/Fail per step
- Regressions Observed
- Remaining Risks / TODO

If something fails, include exact error messages or screenshots and point to suspected files/PR sections.
