# Debug Review Prompt (T-0101)

Please review the supplied code/docs for defects and report in Markdown with:
- Severity: High / Medium / Low
- Steps to Reproduce
- Expected Behavior
- Actual Behavior
- Suspected Files or Modules
- Evidence (logs, screenshots, RAW links if available)

Keep the report concise and actionable; avoid speculative fixes. Use the provided handoff zip/RAW links as the source of truth.
