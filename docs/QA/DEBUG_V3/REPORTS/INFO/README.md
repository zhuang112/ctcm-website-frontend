# INFO - Cross-Task Reference Information

This folder contains **cross-task reference information** that is not specific to any single task but useful for multiple tasks or general project operations.

## What belongs here

- **Cost/Fee information** - API pricing, service costs, budget tracking
- **Tool capability comparisons** - AI model features, library comparisons
- **Environment notes** - Setup guides, configuration references
- **General reference docs** - Anything useful across multiple tasks

## What does NOT belong here

- Task-specific outputs (use `docs/QA/DEBUG_V3/REPORTS/<TASK_ID>/`)
- Decision memos (use `docs/QA/DEBUG_V3/REPORTS/<TASK_ID>/DECISIONS/`)
- Execution logs (use `docs/QA/DEBUG_V3/REPORTS/<TASK_ID>/EXECUTION_LOG.md`)

## Naming convention

```
<TOPIC>_<YYYYMMDD>.md
```

Example: `Claude_API_Fee_Information_20251219.md`
